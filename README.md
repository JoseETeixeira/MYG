# MYG


## Compiling


- UBUNTU

Install GLAD 

```
pip install glad
sudo apt-get install zlib1g-dev
sudo apt-get install libpng-dev
cd third_party/boost 
./bootstrap.sh 
./b2 install --prefix=/usr/share/Boost

```


Windows

```
pip install glad
cd third_party/boost 
./bootstrap.bat 
./b2 install --prefix=C:\boost

```
